# 📏 Conventions

## Mục tiêu
- Đảm bảo **codebase nhất quán**, dễ đọc, dễ review.  
- Giúp dev mới “hòa nhập nhanh” vào cách làm việc của team.  
- Tránh những “cuộc tranh luận bất tận” về style.
